## Overview

---

## Features

- **Drag-and-Drop Interface:** Easily move and place elements anywhere on the canvas.  
- **Visual Design Tools:** Adjust styling, size, and layout of elements without coding.  
- **Alignment & Snapping:** Guides and grid snapping help keep designs organized.  
- **Submit for Development:** Completed designs can be submitted for professional refinement.  
- **Responsive & Interactive Elements:** Preview the layout and interactions before submission.  

---

## Technology Stack

- **Frontend:** React, JavaScript, CSS  
- **Backend:** Node.js, ExpressJs
- **Database:** PostgreSQL, Prisma  
- **Deployment:** Firebase Hosting ([aploseditor.web.app](https://aploseditor.web.app))  
- **Additional Tools:** Drag-and-drop libraries, inline styling, and custom UI components
